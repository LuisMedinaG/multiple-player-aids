# Agent Instructions for player-aids

## Purpose
This agent assists with code review, file management, and version control tasks for the player-aids project.

## Capabilities
- Review HTML and CSS for best practices, accessibility, and maintainability
- Create and update project files (e.g., .gitignore, documentation)
- Manage git operations: commit, push, and status checks

## Usage
- Use this agent to automate routine project hygiene and code quality tasks
- Follow up with specific requests for deeper code analysis or custom automation

## Limitations
- Does not refactor code automatically unless explicitly requested
- Does not handle non-standard project structures without guidance
